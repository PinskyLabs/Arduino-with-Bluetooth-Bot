## Info
We made this bot for IIEST's RoboDarshan's ROBOCON competition where we competed in Death Race and Soccered Games.

## Components:
- 2 L298N Motor Drivers
- 4 Johnson Grade-B motors
- 1 Arduino UNO
- 1 HC-05 Bluetooth Module
